/*------ О проекте ------*/
J2DS - Игровой 2D движок для написания браузерных игр 
на основе HTML5 технологии.

Жанровая направленность: любая
Пространство: 2D
Лицензая: Полностью бесплатно
Исходный код: Открыт


/*------ J2DS v. 0 ------*/
= Сцены
- Поддержка множества сцен
- Очистка сцены
- Отрисовка сцены
- Ограничение FPS
- Игровые циклы
- Поддержка клавиатуры
- Поддержка мыши
- Слежение за объектами
- Перемещение "камеры"
- Скролл камеры
- Проецирование координат мыши в игровые
- Кликабельность объектов
- Константы клавиш
- Поддержка тачскрина для Android/iPhone
- Работа с множественными циклами

- Невидимые объекты
- Движение/Позиционирование

- Расчет дистанции до объекта
- Столкновения объектов
- Примитивы
- Проверка видимости в пределах активной сцены
- Работа с текстом
